# linux-post-install

A post install bash script to install the following programs for a freshly installed ubuntu:
- vlc
- vs-code
- pipx
- yt-dlp


Steps to use:

1. Go to home folder of ./post-install.sh
> cd ~/<folder>/

2. Make ./post-install.sh executable
> chmod +x install_apps.sh

3. Run the script
> ./install_apps.sh
